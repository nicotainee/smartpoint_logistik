// Cara Kerja

Simpel. Cepat. Presisi.

Dari daftar alamat sampai kurir navigasi, semua otomatis dalam hitungan detik.

01
📝
Input Alamat
Admin input nama penerima, pin lokasi di peta, dan patokan visual rumah. Foto bisa diupload langsung.
02
⚡
QR Otomatis
Sistem langsung generate QR Code unik yang terhubung ke koordinat GPS presisi lokasi penerima.
03
📦
Tempel ke Paket
QR Code dicetak dan ditempel ke paket. Satu QR untuk satu alamat penerima secara permanen.
04
🗺️
Kurir Navigasi
Kurir scan QR → muncul foto rumah + patokan visual → klik navigasi → langsung ke lokasi tepat.

Dibangun untuk skala besar.

Semua yang dibutuhkan jasa logistik modern untuk operasional yang efisien.

📍
Pin Lokasi di Peta
Input alamat cukup klik peta — tidak perlu copy-paste koordinat. Akurat sampai level meter.
📷
Foto Rumah
Upload foto pagar atau tampak depan rumah. Kurir langsung tahu mana rumahnya sebelum tiba.
⚡
QR Generate Otomatis
Setiap alamat baru langsung punya QR Code — bisa langsung diprint dan ditempel ke paket.
🗂️
Dashboard Admin
Kelola semua Smart Point dari satu dashboard — tambah, edit, hapus, dan cari alamat dengan mudah.
🔌
Siap Integrasi API
Sistem logistik yang sudah ada bisa integrasi via API — input data massal tanpa perlu manual.
📱
Mobile Friendly
Halaman kurir dioptimalkan untuk HP — scan QR, lihat info, navigasi. Semua dalam satu layar.
